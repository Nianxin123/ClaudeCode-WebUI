# Claude Code 本地Web UI 技术方案

## 需求分析
1. 本地浏览器访问的Web页面
2. 支持与AI对话
3. 支持上传文件和图片
4. 连接本地部署的Claude Code

## 技术架构

### 前端
- **技术栈**: HTML5 + CSS3 + Vanilla JavaScript
- **UI框架**: 自定义聊天界面
- **功能**:
  - 对话输入框
  - 消息展示区（支持Markdown渲染）
  - 文件/图片上传组件
  - 响应式布局

### 后端
- **技术栈**: Node.js + Express
- **功能**:
  - RESTful API 接口
  - 调用 Claude Code CLI
  - 文件上传处理
  - 会话管理

### Claude Code 集成
- **方式**: 通过 child_process 调用 Claude Code CLI
- **命令**: `claude` 或 `claude-code` CLI工具
- **参数**: 支持对话、文件输入等

## 项目结构
```
AI使用工具/
├── server.js          # 后端服务
├── package.json       # 项目配置
├── public/            # 前端静态文件
│   ├── index.html     # 主页面
│   ├── style.css      # 样式
│   └── app.js         # 前端逻辑
├── uploads/           # 上传文件目录
└── 技术方案.md        # 本文档
```
